What is Sugar Snake?

Sugar Snake is a fun multiplication practice game.

You play a snake eating yummy "multapples" with the goal to be the largest snake ever!

But watch out! Eating the wrong answers will send you off course!

Our math-minded snake

Using Sugargame - Starting Troubles

Sugargame, a variant of pygame made specifically for use with the XO machines came with a few difficulties starting up and along the way like:

  • Keyboard controls
  • Menu buttons
  • Events handling with Sugar GTK

Difficulties - What we would have done differently

  • Getting setup was a little difficult especially getting our code to integrate into a Sugar Activity
  • We had a few issues sticking to our process of branching and then merging, but were able to re-group onthat

What went well

  • Pretty good mix of skills in the team
  • Was able to successfully understand and be able to alter the pre-existing Planked code to our needs

Given more time...

  • Difficulty levels to allow for varying degrees of math problems
  • Testing different implementations of the game, i.e. eating wrong problem would reduce size, not be a game over
  • Create a nice icon for the Sugar OS to recognize as an activity
